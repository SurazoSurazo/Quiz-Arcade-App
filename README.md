# Quiz Quest

## 概要

Quiz Quest は、ゲームセンターのアーケードゲーム風に作成したクイズアプリです。

プレイヤーは「予選」と「本選」の2ステージを進み、制限時間内に問題へ回答します。正解数だけでなくスピードボーナスによる得点要素もあり、最終的なスコアをランキングへ登録できます。

主な機能:

- 予選・本選のステージ制
- 制限時間付きクイズ
- スピードボーナス機能
- ランキング機能
- 復習モード
- 間違えた問題の記録
- 復習履歴の保存
- 複数の問題形式対応

## 問題形式

このアプリでは複数の出題形式に対応しています。

### 選択問題

複数の選択肢から回答を選択します。

### 入力問題

答えを直接入力します。

### 並び替え問題

項目を正しい順番に並べ替えて回答します。

### 複数選択問題

複数の正解を選択して回答します。

---

## ゲームルール

### 予選

- 全5問
- 3問以上正解で本選進出

### 本選

- 全10問
- 8問以上正解でゲームクリア

### 得点ルール

基本点:

- 正解 → 1点

スピードボーナス:

- 選択問題: 3秒以内
- 入力問題: 5秒以内
- 並び替え問題: 5秒以内
- 複数選択問題: 5秒以内

ボーナス獲得時:

- 2点獲得

## インストール

### リポジトリを取得

```bash
git clone https://github.com/ユーザー名/quiz-arcade.git
```

### プロジェクトへ移動

```bash
cd quiz-arcade
```

### ローカルサーバー起動例

PHP 内蔵サーバー:

```bash
php -S localhost:8000
```

```text
http://localhost:8000
```

## 使い方

### ゲーム開始

ホーム画面から

```text
PRESS START
```

を押して開始します。

### クイズ回答

問題形式に応じて回答します。

- 選択
- 入力
- 並び替え
- 複数選択

### 結果確認

ゲーム終了後:

- 正解数
- 獲得点数
- ランキング順位

を確認できます。

### 復習機能

間違えた問題は保存され、後から復習できます。
